## 参考借鉴

- 登录、关于、课表、电费查询界面借鉴参考使用[We重邮](https://github.com/mcc108/wecqupt)

- 校园导航路线规划使用[高德导航API](https://lbs.amap.com/)

- 成绩查询界面统计图表[wx-charts](https://github.com/xiaolin3303/wx-charts)

- [ColorUI](https://github.com/weilanwl/ColorUI)
